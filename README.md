
shadow-cljs module loader example
---

### Usage

```bash
yarn shadow-cljs compile app
yarn html
yarn serve
```

Or in release mode:

```bash
yarn shadow-cljs release app
yarn html-dist
yarn serve-dist
```

Based on https://github.com/thheller/shadow-cljs/wiki/ClojureScript-for-the-browser#module-loader
